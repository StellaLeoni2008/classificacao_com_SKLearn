## 📌 Sobre o Projeto
Este notebook foi desenvolvido durante meus estudos de Machine Learning utilizando Python e a biblioteca Scikit-learn. Ao longo do projeto, explorei conceitos fundamentais de classificação, separação de dados para treino e teste, avaliação de modelos e Support Vector Machines (SVM).

O objetivo principal foi compreender como modelos de aprendizado supervisionado funcionam na prática, desde exemplos simples até aplicações com conjuntos de dados mais realistas.

---

## 📚 Conceitos Praticados

**🔹 Introdução à Classificação**
- Criação de modelos de classificação
- Treinamento de algoritmos supervisionados
- Uso de features e labels
- Primeiras previsões com Machine Learning

**🔹 Separação de Dados**
- Divisão entre treino e teste
- Importância da validação dos modelos
- Uso do `train_test_split`
- Controle de aleatoriedade com `random_state`

**🔹 Métricas de Avaliação**
- Cálculo de acurácia
- Comparação entre previsões e resultados reais
- Avaliação de desempenho dos modelos

**🔹 Baseline**
- Criação de modelos baseline
- Comparação entre modelos simples e modelos treinados
- Entendimento da importância de um ponto de referência

**🔹 Support Vector Machine (SVM)**
- Implementação com `LinearSVC` e `SVC`
- Conceito de separação linear
- Não linearidade em Machine Learning
- Ajuste de parâmetros como `gamma`

**🔹 Visualização de Dados**
- Exploração de datasets
- Gráficos para análise visual
- Interpretação de padrões nos dados
